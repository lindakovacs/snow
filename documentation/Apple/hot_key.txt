В меню Mac OS X встречаются перечисленные ниже клавиши-модификаторы:

Command (клавиша Command) — на некоторых клавиатурах Apple на этой клавише также содержится логотип компании Apple (логотип apple)
Control (клавиша Control)
Option или Alt (клавиша Option) — на этой клавише также может быть написано Alt
Shift (клавиша Shift)
Caps lock (Caps Lock) — включает или отключает режим ввода заглавных букв
fn (клавиша Function)

Xcode

Сmd+Shift+J  сразу показать его в дереве файлов
Ctrl+Shift+E переименование переменной в текущем скопе

Ctrl+A – в начало строки
Ctrl+E – в конец строки
Ctrl+K – удалить строку

esc – спрятать (показать) меню предложений
Tab – применить предложение

Cmd + Y – активировать (деактивировать) точки остановки
Cmd + Control+Y – продолжить компиляцию (в дебаггере)
F6 – степ-овер

Cmd + двойной щелчок - переключение к определению идентификатора
0
Opt + двойной щелчок - найти документацию для идентификатора
Tab + → или Ctrl + / - выбрать следующий аргумент в автозаполнении
Shift + Tab + → или Shift + Ctrl + / - выбрать предыдущий аргумент в автозаполнении
Ctrl + Shift + . - в обратном направлении между вариантами автозаполнения
Opt + Cmd + двойной щелчок на идентификаторе - найти в документации
Cmd + двойной щелчок на объекте в Document Window - переключиться к классу в XCode из IB
Ctrl + Shift + щелчок на объекте - выбрать наложенные друг на друга элементы в Interface Builder

Cmd + Shift + E - скрыть список файлов над редактором кода
Cmd + Opt + Shift + E - скрыть список файлов над редактором кода и «Groups and Files»
Alt + Shift + Cmd + K - очистить папки билда (build)
cmd + стрелки влево/вправо - поворот симулятора


Панель навигации


Показать редактируемый файл в панели навигации — Shift + Cmd + J;
Открыть глобальный поиск — Shift + Cmd + F (Обычный поиск  Cmd + F ищет введённый текст только в открытом файле, но что делать, если вам понадобится поиск по всем файлам?);


Очистить папки билда (build) — Alt + Shift + Cmd + K;

Поворот симулятора — cmd + стрелки влево/вправо;
Открыть файл в новом окне — Shift + Cmd + T;


Редактор кода

Поиск по файлу — Cmd + F;
Переместить курсор в начало строки — Ctrl +A;
Переместить курсор в конец строки — Ctrl +E;
Поменять местами символы, прилегающие к курсору — Ctrl +T;
Удалить строку — Ctrl +K;
Спрятать (показать) меню предложений — Esc;
Применить предложение — Tab;

ВЫДЕЛЕНИЕ:
Shift-Command-стрелка вправо Выделить текст с точки ввода до конца текущей строки (*)
Shift-Command-стрелка влево	 Выделить текст с точки ввода до начала текущей строки (*)
Shift-стрелка вправо	Расширить область выделения текста на один символ вправо (*)
Shift-стрелка влево	    Расширить область выделения текста на один символ влево (*)
Shift-Command-стрелка вверх	Выделить текст с точки ввода до начала текущего документа (*)
Shift-Command-стрелка вниз	Выделить текст с точки ввода до конца текущего документа (*)
Shift-стрелка вверх	Расширить область выделения текста на одну строку вверх до границы ближайшего символа в текущем горизонтальном положении точки ввода (*)
Shift-стрелка вниз	Расширить область выделения текста на одну строку вниз до границы ближайшего символа в текущем горизонтальном положении точки ввода (*)
Shift-Option-стрелка вправо	Расширить область выделения текста до конца текущего слова, при повторном нажатии — до конца следующего слова (*)
Shift-Option-стрелка влево	Расширить область выделения текста до начала текущего слова, при повторном нажатии — до начала следующего слова (*)
Shift-Option-стрелка вниз	Расширить область выделения текста до конца текущего абзаца, при повторном нажатии — до конца следующего абзаца (*)
Shift-Option-стрелка вверх	Расширить область выделения текста до начала текущего абзаца, при повторном нажатии — до начала следующего абзаца (*)

Control-Пробел	Переключение между текущим и предыдущим источником ввода
Option-Control-Пробел	Переключение между всеми задействованными источниками ввода
Option-Command-Esc	Принудительный выход
