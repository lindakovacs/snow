iOS App ID идентификатор приложения (строка), который, в основном, используется в provisioning profile,
он определяет, какие приложения могут быть подписаны и выполнены.

- Регистрация App ID с помощью XCode
При использовании Xcode, вы получите App ID по умолчанию. Этот идентификатор приложения может быть использован для всех ваших iOS приложений, которые не требуют услуг, таких как Push-уведомления,
ICloud, Game Center, In-App Purchase, Data Protection, and Wallet (кошелек).

- Register App IDs Manually
Team agents and admins can manually register App IDs and enable them for app services.

HOTKEYS:

Фыделенный фрагмент. Сдвиг вправо – command+], сдвиг влево – command+[.

Навигатор - верхняя строка с файловой иерархией.
Counterparts - одноимённые файлы с разными расширениями (например, m и h).
Если нам понадобятся 2 файла, не парных (не Counterparts), то нам необходимо кликнуть нужный файл в навигаторе с зажатой клавишей Alt.
!!! Далее нажимаем на Manual правого окна и нажимаем Counterparts
В правом верхнем углу 2 окна есть маленький плюсик, который добавляет еще одно окно редактирования – Assistant Editor.

Vlew->Navigators->Show Symbol Navigator  просмотр иерархии классов
